Overview

Guard Llama

Role: UI Designer

Client: Guard Llama

Guard Llama is a physical device that utilizes smartphone's GPS location to notify the nearest dispatch in emergency situations. With the average 40 minutes response time, Guard Llama is able to speed up the process up to 70% faster.


I worked on this project as the UI and interaction designer, in collaboration with 2 other UI designers and my project director to redesign Guard Llama’s landing page and the accompany mobile app.

Tools
Methods

Adobe Illustrator, mood-boarding, exploratory style tiles, style guide, competitive analysis, animations.

Project Brief

My client intially asked for UI / IxD design of their mobile application; however, with our reevaluation, we decided to redesign the product landing page as well to project a cleaner branding and help explain the product to drive up the revenue.

Some of the biggest challenges were to differentiate GuardLlama from the multitude of existing personal safety applications and including additional helpful features without distracting the users from the key function of the application.

Style Exploration

Mood Boarding

After a quick UI competitve analysis for the different apps out there, I created a mood board to gather some inspirations.

Many personal safety products are designed based on selling fear; we decided to differentiate GuardLlama by being friendly and focus on treasure the moments with our loved ones, yet retaining the professional and dependable feel.

Key Findings: The client expressed his liking of the Nest Cam website because it is able to make a surveillance system feel friendly and natural.

Style Tiles

After mood boarding, I created three distinct style tiles to gather intial feedback from the client as style exploration.

Key Findings: The client last style tile's direction the best, so I proceeded the rest of the project with this style.

Style Guide

Typography

I chose DIN because it is a font that combines techy-modern and humanist to achieve that friendly yet modern and dependable feel.

Brand Color

I used a light blue for the primary color because it has a friendlier vibe and feels less corporate than a darker blue.

Key Interaction

The most important function of the app is to call for help. I made sure that the minimized "Call For Help" button on any given screen remains at the same placement of the button on the "Call For Help" screen.

Landing Page

After showing my client the redesign of the mobile application, the client asked for a redesign of their current landing page with the same style of the mobile application to maintain consistent branding across platforms.

Responsive Design

It is very important to allow the users to be able to access the webpage at the locations they desire, therefore I made sure the design of the landing page responds well to devices of different screen sizes.

Results

Before & After

The client was very satisfied with the redesign results. The redesigned landing page and mobile application maintained a much more consistant style across platforms, which enahnced the product’s branding and professional feel.
